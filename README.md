# Voice-Enabled-Hospital-Token-Management-System-using-Python
A Python-based hospital token management system with voice-assisted input for elderly patients, webcam photo capture, Excel-based daily record storage, and automatic PDF receipt generation. The system supports 24-hour operation, improves patient registration efficiency, and reduces manual workload in hospital environments.
